## SpiderScrapyArticle

### 介绍
* 基于基于scrapy的最简爬虫工程
* 目前以实现爬取的网站为-每日一文
* 写完爬虫后，发现每日一文本身就有对外提供的API[哭脸]


### 安装使用
* 初始化数据参考SpiderScrapyArticle工程的sql初始化文件
* 直接执行get_meiriyiwen_api.py(50行代码)，4分钟获取2500+优质文章
* 忽略z1目录下的scrapy框架吧
